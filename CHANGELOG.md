# Changelog

All notable changes to this project will be documented in this file.

## [1.7] - 13.05.2026

### Added
- Option to set custom font sizes for Bible verses in both the app and the widget.
- Automatic proportional scaling for verse references to maintain visual hierarchy.
- Immediate widget refresh when changing font size or color settings.
- German and English translations for all new appearance settings.

### Changed
- Improved settings UI with a new "Appearance" category.
- Improved App View on modern Devices with Notch
- Updated project to Version 1.7 (Build 8).

## [1.2] - 25.10.2023

### Added
- Widget Update now working also when changing bible verse version in settings

### Fixed
- Fixed some Crashes on some Devices which PlayStore reported

## [1.0] - 25.10.2023

### Added
- Everything! This is the first release.
